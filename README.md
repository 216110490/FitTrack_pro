# FitTrack Pro

FitTrack Pro is a Flutter fitness tracking app created for the IS487 project.

## Phase 3 Features
- Responsive Flutter UI with bottom navigation
- Provider state management
- SQLite local database for workout CRUD
- REST API integration for fitness tips
- Native device image picker for profile progress photo
- Light and dark mode
- Basic unit test

## How to Run
1. Run `flutter pub get`
2. Run `flutter run`

## Notes
The app is made as a student project and focuses on demonstrating the required course concepts.
